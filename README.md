# uAgents Hack

A lightweight, autonomous agent built using uAgents to facilitate intelligent project scaffolding/initialisation and automation.

## Development Setup

### uAgents

Install dependencies

```bash
cd agent
pipenv install
```

Run agent

```bash
pipenv run python -m src.agent
```

### UI

Install dependencies

```bash
cd ui
npm install
```

Run dev server

```bash
npm run dev
```
